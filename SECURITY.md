# Security Policy

## Supported Versions

The `main` branch is the actively supported development branch.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use GitHub private vulnerability reporting if it is enabled for this
repository. If private reporting is not available, contact the maintainer
directly.

Include:

- Affected version or commit.
- Steps to reproduce.
- Potential impact.
- Suggested fix, if known.

## Safety Scope

AutoPulse is designed for read-only OBD-II telemetry analysis. Reports involving
diagnostic write/control services, credential exposure, unsafe adapter behavior,
or malformed replay data crossing validation boundaries are security-sensitive.
